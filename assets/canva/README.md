## Export Note

The Canva connector available in this Codex session created editable Canva
designs and resized derivatives, but it did not expose a full-resolution PNG
export action. The public repo still includes production-sized PNG assets from
the public-safe Black2Africa visual package. If exact Canva rasters are needed,
export these Canva designs manually as PNG files and overwrite the matching
public repo paths listed above.

Do not export or upload private screenshots, legal records, account consoles,
source-code screens, deployment receipts, customer records, credentials, or
private workflow diagrams.
