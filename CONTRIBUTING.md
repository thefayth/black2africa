# Contributing

Black2Africa is not accepting public code contributions through this protected
surface.

You may suggest corrections to public documentation, accessibility notes,
privacy-boundary improvements, or partnership ideas through an approved contact
path.

Do not submit:

- source code for the private engine
- deployment scripts or production configuration
- credentials, secrets, tokens, or account screenshots
- private prompts, workflows, receipts, queues, logs, or datasets
- customer, legal, medical, benefits, family, housing, or private creative data
- copyrighted third-party material you do not control

Any submitted suggestion may be rejected, edited, archived, or moved into a
private review process. Submission does not create ownership transfer,
compensation, employment, partnership, or a license to use protected project
materials.
