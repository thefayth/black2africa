GitHub export. Only assets rated `PUBLIC SAFE` were copied into

## Rating Key

- `WORDPRESS ONLY`: usable on FaithCheltenham.com or black2africa.xyz, but not
  copied into this GitHub export.
- `PRIVATE / DO NOT USE`: excluded from the public export.
