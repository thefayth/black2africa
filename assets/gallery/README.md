# Gallery Assets

This folder contains public-safe Black2Africa visual assets copied from
`public/brand/black2africa/` and public-facing project visual assets from the
Black2Africa WordPress plugin asset folder after inspection.

These assets are intended for GitHub README support, FaithCheltenham.com draft
composition, and public project storytelling. They are not private screenshots,
account-console captures, legal records, customer records, or implementation
files.

See `docs/IMAGE_ASSET_AUDIT.md` before publishing.
